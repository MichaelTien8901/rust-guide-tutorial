# Part 1: Getting Started

Welcome to Rust! This part will help you set up your development environment and write your first Rust program.

## What You'll Learn

- What Rust is and why it's worth learning
- How to install Rust on your system
- Setting up VS Code or your preferred editor
- Creating and running your first Rust program
- Basic Cargo commands for project management

## Chapters

1. [Introduction to Rust]({% link part1/01-introduction.md %}) - What is Rust and why learn it?
2. [Installation]({% link part1/02-installation.md %}) - Install Rust on Linux, macOS, or Windows
3. [IDE Setup]({% link part1/03-ide-setup.md %}) - Configure VS Code with rust-analyzer
4. [Alternative Editors]({% link part1/04-alternative-editors.md %}) - IntelliJ, Neovim, Emacs, and more
5. [Docker Setup]({% link part1/05-docker-setup.md %}) - Containerized development environment
6. [Hello World]({% link part1/06-hello-world.md %}) - Your first Rust program
7. [Cargo Basics]({% link part1/07-cargo-basics.md %}) - Project management with Cargo

## Prerequisites

- A computer running Linux, macOS, or Windows
- Basic command line knowledge
- About 1-2 hours to complete this section

## Quick Path

If you're in a hurry:
1. [Install Rust]({% link part1/02-installation.md %})
2. [Write Hello World]({% link part1/06-hello-world.md %})

Then come back for the rest when you have time!
